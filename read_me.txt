.how to run the project :-

step-1 : unzip the folder in your local computer.

step-2 : you have to initialize the npm init in the project folder where you have unzipped the folder.

step-3 : now run the node index.js file in the terminal.

step-4 : and now your live server will be runnning 

step-5 : open the browser and search "localhost:{port_no}/"

step-6 : it will redirect you to the home page
